# 관계부 — HTML 프로토타입

샘플 데이터가 들어 있는 단일 HTML 목업입니다. 빌드/서버 없이 브라우저에서 바로 열어 모든 화면을 둘러볼 수 있습니다.

## 사용법

```bash
# macOS
open prototype/index.html

# 또는 더블클릭
```

또는 dev server에 의존하지 않고 정적으로 띄우려면:

```bash
npx serve prototype
# → http://localhost:3000 (또는 5000 등)
```

## 포함된 화면

| 라우트 | 설명 |
|--------|------|
| `#login` | 로그인 + 게스트 시작 |
| `#home` | 홈 (인물 5명, 리마인더 3건, AI 추천 카드) |
| `#person/<id>` | 인물 상세 (프로필, 점수, 액션, 타임라인, 메모) |
| `#person-new` | 인물 추가 폼 |
| `#reminders` | 리마인더 (지난/오늘/이번주/이후 버킷) |
| `#search` | 검색 + 관계 유형 필터 |
| `#settings` | 설정 (프로필, AI 프로바이더, 테마, 데이터 export) |

## 동작하는 인터랙션

- 화면 간 라우팅 (해시 기반)
- 하단 탭 네비게이션 (5탭, FAB 강조)
- 검색 + 관계 유형 필터 토글
- 라이트/다크/시스템 테마 전환
- JSON export 다운로드
- 인물 카드 → 상세 진입
- 리마인더 → 인물 상세 진입

## 목업이라 동작하지 않는 것

- 실제 데이터 저장 (alert로만 피드백)
- 인물 추가 폼 제출 (UI만, 저장 안 됨)
- 카카오/구글 OAuth
- 실제 AI 호출
