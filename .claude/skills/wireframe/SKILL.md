# 와이어프레임 스킬

## 산출물 형식 (모바일 360x640 기준)

```
┌─────────────────────────────┐
│  ← 인물 상세            ⋮  │  ← 헤더 (back, 메뉴)
├─────────────────────────────┤
│   👤  김지수                │
│   친구 · MBTI: ENFP         │
│   마지막 연락: 12일 전      │
├─────────────────────────────┤
│  [💬 연락 기록] [🎁 선물]   │  ← 액션 칩
├─────────────────────────────┤
│  AI 관계 건강도            │
│  ████████░░  78점          │
│  "최근 연락이 뜸합니다..."  │
├─────────────────────────────┤
│  타임라인                   │
│  • 2026-04-20 카톡          │
│  • 2026-03-15 생일선물 5만원│
└─────────────────────────────┘
[홈] [검색] [+] [리마인더] [설정]  ← 하단탭
```

## 화면별 표준 구성

- **헤더**: 좌(뒤로/메뉴), 중(타이틀), 우(액션)
- **본문**: 스크롤 영역, 카드/리스트 단위
- **하단탭**: 5개 이내 (홈/검색/추가/리마인더/설정)
- **FAB**: 추가 액션은 우하단 + 버튼

## 작성 규칙

- 화면당 별도 코드블록 (h2 제목)
- 각 박스 옆에 `← 설명` 주석
- 데이터 placeholder는 한국어 예시값
- 인터랙션은 화면 아래 "Actions" 섹션에 명시
- 모바일 → 데스크톱 변형 별도 표기
