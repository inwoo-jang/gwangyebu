> 작성자: UX Designer Agent
> 최종 갱신: 2026-05-04
> 버전: v0.1 (M1 우선, M2 일부 포함)

## 2. 네비게이션 구조도

```
[비로그인]
  └ 랜딩 ─→ 가입/로그인 ─→ 온보딩 1·2·3 ─→ 홈

[로그인 후 — 5탭 BottomNav]
  ┌────────────────────────────────────────────────────┐
  │ [🏠 홈] [🔍 검색] [➕ 추가] [🔔 리마인더] [⚙ 설정] │
  └────────────────────────────────────────────────────┘

  홈
  ├ AI 추천 위젯 ─→ AI 분석 결과 화면 (Top 5 카드)
  │                  └ 인물 상세 (deep link)
  ├ 오늘의 리마인더 ─→ 리마인더 리스트
  ├ 인물 카드 리스트 ─→ 인물 상세
  │                       ├ 연락 기록 모달
  │                       ├ 경조사/선물 탭 (M2)
  │                       └ 인물 편집 폼
  └ FAB(+) ─→ 인물 추가 폼

  검색
  └ 결과 카드 리스트 ─→ 인물 상세

  + (추가 FAB 탭)
  └ 바텀시트: [직접 입력] [연락처에서 가져오기(M2)] [명함 스캔(M2)]
        └ 인물 추가 폼

  리마인더
  ├ 오늘 / 이번주 / 다가올 / 완료 탭
  ├ + 리마인더 추가 ─→ 리마인더 생성 모달
  └ 항목 탭 ─→ 인물 상세 (action=log)

  설정
  ├ 알림
  ├ AI 프로바이더
  ├ 데이터 (Export / Import / 휴지통)
  ├ 계정 (표시명·이메일·비밀번호·탈퇴)
  └ 약관/개인정보/오픈소스
```

## 4. 글로벌 레이아웃 룰

### 4.1 모바일 (360~430px)
```
┌─────────────────────────────────┐
│  헤더 (56px, sticky)            │ ← back/타이틀/액션
├─────────────────────────────────┤
│                                 │
│  본문 스크롤 영역               │ ← 안전영역 패딩 px-4 (16)
│  (BottomNav 80px + safe-area    │
│   bottom 만큼 padding-bottom)   │
│                                 │
├─────────────────────────────────┤
│  BottomNav (64px + safe area)   │ ← 5탭, fixed bottom
└─────────────────────────────────┘
```

- 본문 콘텐츠 max-width: 100%, 좌우 패딩 16px
- 헤더/하단탭은 `backdrop-blur` + 반투명 (스크롤 시 투명도 유지)
- FAB(+) 위치: 우측 하단, BottomNav 위 16px 떨어진 곳 (홈 화면 한정)

### 4.2 데스크톱 (≥ 1024px)
```
┌──────────────────────────────────────────────────────────────┐
│  헤더 (검색바 + 사용자 메뉴)                                  │
├──────────────┬───────────────────────────────────────────────┤
│              │                                               │
│  사이드 네비 │   메인 콘텐츠 (max-w-3xl center)              │
│  (240px)     │                                               │
│  · 홈        │                                               │
│  · 검색      │                                               │
│  · 추가      │                                               │
│  · 리마인더  │                                               │
│  · 설정      │                                               │
│              │                                               │
└──────────────┴───────────────────────────────────────────────┘
```

- 모바일 BottomNav 대신 좌측 SideNav (lg: 이상)
- 인물 상세는 1024+ 에서 좌(리스트 220px) / 우(상세) 2-pane 가능 (M1는 단일 pane)
- 모달/Sheet은 모바일에서 풀스크린 Sheet, 데스크톱에서 Dialog

## 5. 공통 상태 (모든 화면 공통)

| 상태 | 처리 |
|------|------|
| 로딩 | 스켈레톤 (3개 카드 placeholder), 5초 초과 시 "느려지고 있어요" |
| 빈 상태 | 일러스트 + 안내 카피 + Primary CTA |
| 에러 | 카드 형태의 에러 안내 + 재시도 버튼 + 토스트 |
| 오프라인 | 상단 노란색 배너 "오프라인이에요. 연결되면 동기화됩니다" |
| 인증 만료 | 자동 갱신 → 실패 시 로그인 화면 + 원래 URL 복귀 |

## 6. 포커스 / 접근성

- 모든 인터랙티브 요소 최소 44x44 터치 타겟
- 포커스 링: `ring-2 ring-ring ring-offset-2`
- 색 대비: WCAG AA (텍스트 4.5:1 이상)
- 다크모드 자동 + 수동 토글 (설정)
- 한국어 라인 높이는 영문보다 1.6배 권장 (`leading-relaxed` 기본)

## 7. 핵심 인터랙션 패턴

| 패턴 | 사용처 | shadcn 컴포넌트 |
|------|--------|-----------------|
| 풀스크린 모달 (모바일) | 폼 입력, 연락 기록 | `Sheet` (side="bottom" + h-full) |
| 다이얼로그 (데스크톱) | 폼 입력, 확인 | `Dialog` |
| 바텀시트 (옵션 선택) | 추가 방식, 정렬, 필터 | `Sheet` (side="bottom") |
| 토스트 (피드백) | 저장 완료, 오프라인 안내 | `Toast`(추가 설치 필요) |
| 칩 / 태그 | 관계유형, 태그, 액션 | `Badge` 변형 |
| 리스트 카드 | 인물, 리마인더, 추천 | 커스텀 컴포넌트 (PersonCard 등) |
| 탭 | 인물 상세, 리마인더 | `Tabs` |
| 검색바 | 홈 상단, 검색 화면 | `Input` + 커스텀 |
