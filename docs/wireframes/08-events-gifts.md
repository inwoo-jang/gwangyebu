## 8.7 인터랙션 / 액션

| 요소 | 동작 |
|------|------|
| FAB(+) | 8.2 추가 Sheet |
| 경조사 종류 칩 | 단일 선택, 생일 선택 시 "매년 알림" 자동 ON |
| 금액 입력 | 천단위 콤마 자동 표시, 소수점 차단 |
| 참석 = 불참 + 금액 0 | "마음만 보냄"으로 자동 표기 (메모 자동 채움 가능) |
| 연결된 경조사 (선물) | 같은 인물의 ±7일 내 Event 자동 추천 |
| 항목 탭 | 편집 Sheet |
| 항목 스와이프 좌 | 삭제 (확인) — "이 항목을 삭제하면 합계에서 제외됩니다" |
| 합계 카드 탭 | 통계 페이지 (8.5) 이동 |
| 캘린더 동기화 (M3) | 토글 ON 시 구글 캘린더에 일정 추가 |
| 생일 자동 리마인더 | event_type=birthday + repeat=yearly 시 매년 알림 자동 생성 |

## 8.8 사용 shadcn 컴포넌트

| 영역 | 컴포넌트 |
|------|----------|
| 탭 | `Tabs`, `TabsList`, `TabsTrigger` |
| 합계 카드 | `Card` + 커스텀 |
| 종류 칩 | `Badge` 또는 `ToggleGroup` |
| 폼 | `Form`, `Input`, `Textarea`, `RadioGroup` |
| 날짜 | `Calendar` + `Popover` |
| 추가 Sheet | `Sheet` side="bottom" |
| 자동 매칭 (Event) | 커스텀 Combobox |
| 차트 (통계) | recharts (또는 chart.js, 별도 설치) |

## 8.9 데이터 의존성

| 영역 | 엔티티 |
|------|--------|
| 경조사 리스트 | `Event where person_id={id} order by occurred_at desc` |
| 선물 리스트 | `Gift where person_id={id} order by occurred_at desc` |
| 합계 카드 (이번 해) | `sum(case when direction='received' then amount else 0 end)`, `sum(case when direction='sent' then amount else 0 end)` (당해 연도) |
| 통계 (Top 5) | `Gift group by person_id order by sum(amount) desc limit 5` |
| 연결 추천 | `Event where person_id={id} and abs(occurred_at - $gift_date) ≤ 7 days` |

## 8.10 상태별 처리

| 상태 | 화면 |
|------|------|
| 항목 0건 | "아직 기록이 없어요. 첫 경조사를 추가해볼까요?" + [+ 추가] |
| 합계 0원 | "아직 주고받음 기록이 없어요" |
| 캘린더 미연동 (M3) | 동기화 토글 옆 "[연동하기]" 안내 |
| 미래 날짜 (예정 결혼식) | 허용, 카드에 "예정" 라벨 |
| 1인당 같은 종류 다회 | 허용 (이혼/재혼 등) |
| 통계 데이터 부족 (5건 미만) | "기록이 더 쌓이면 통계가 더 의미 있어져요" |
