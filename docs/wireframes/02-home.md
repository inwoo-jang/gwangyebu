## 2.5 인터랙션 / 액션

| 요소 | 동작 |
|------|------|
| 🔔 헤더 알림 아이콘 | 알림 센터 Sheet 열기 (최근 푸시 이력) |
| ⚙ 헤더 설정 아이콘 | 빠른설정 Sheet (다크모드 토글, 로그아웃) |
| 검색바 (모바일) | `/search` 이동 |
| 검색바 (데스크톱) | 인라인 자동완성 드롭다운 |
| AI 추천 위젯 카드 | `/p/{id}?from=ai` (해당 인물 상세) |
| AI 추천 위젯 더보기 | `/ai/recommendations` |
| AI 카드 [지금 연락] | 채널 선택 바텀시트 → 외부 앱 |
| 리마인더 [완료] | 인라인 ContactLog 모달 → 저장 시 status=done |
| 리마인더 [연기] | Sheet: 1일/3일/1주 옵션 |
| 정렬 ▾ | Sheet (모바일) / Popover (데스크톱) — 최근/이름/추가일 |
| 필터 ▾ | Sheet — 관계유형 / 태그 / 무연락 기간 |
| Person 카드 탭 | `/p/{id}` |
| Person 카드 길게 누름 | 컨텍스트 메뉴 — 방금 연락함 / 리마인더 / 편집 |
| FAB(+) | 바텀시트 — 직접 입력 / 연락처(M2) / 명함(M2) |
| Pull-to-refresh | 데이터 재페치 + 햅틱 피드백 |

## 2.7 데이터 의존성 (data-model.md 참조)

| 구역 | 엔티티 / 쿼리 |
|------|---------------|
| 인사말 요약 | `count(Reminder where date=today + status=active)` |
| AI 추천 위젯 | `Recommendation where week=current order by rank` (Top 5) |
| 오늘 리마인더 | `Reminder where scheduled_at::date = today and status in (active, snoozed)` |
| 최근 연락 리스트 | `Person where deleted_at is null order by last_contact_at desc nulls last limit 50` |
| Person 카드 보조 데이터 | `RelationshipScore.score` (조인) |

가져오기 전략:
- 초기 SSR로 첫 50건 인물 + AI 추천 + 오늘 리마인더 일괄 fetch
- 무한 스크롤은 Client Component (`useInfiniteQuery`)
- AI 추천은 stale-while-revalidate (1시간 캐시)

## 2.8 상태별 처리

| 상태 | 화면 |
|------|------|
| 인물 0명 | 2.2 빈 상태 |
| 인물 1~4명 | AI 추천 위젯 자리에 "조금만 더 등록하면 추천 시작!" 카드 |
| 인물 5명+, 추천 미생성 | "월요일 오전에 첫 추천이 도착해요" 카드 |
| 오프라인 | 헤더 아래 노란 배너 "오프라인 모드 — 캐시된 데이터 표시 중" |
| 데이터 페치 실패 | 에러 카드 + [다시 시도] |
| 무연락 30일+ | Person 카드 우측에 ⚠ 아이콘 + RelationshipScore < 50 시 빨간 점 |

## 2.9 한국어 카피

| 위치 | 카피 |
|------|------|
| 인사말 | 시간대별 — "좋은 아침이에요, {이름}님 ☀" / "오후도 화이팅 💪" / "저녁 어때요? 🌙" |
