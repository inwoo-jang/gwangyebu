## 5.1 모바일 — 리마인더 리스트

```
┌─────────────────────────────┐
│  리마인더              ⋮     │  ← Header (메뉴: 정렬/숨김)
├─────────────────────────────┤
│  [오늘*][이번주][다가올][완료]│  ← Tabs (4개)
├─────────────────────────────┤
│  오늘 · 5월 4일 (월)         │  ← 그룹 헤더
│  ┌───────────────────────┐  │
│  │ 🟠 09:00              │  │  ← Reminder item
│  │ 김부장님 · 고객사      │  │
│  │ "이번 주 추천에 있었어요"│  │
│  │ [✓ 완료] [⏰ 연기]     │  │  ← Actions
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ 🟡 18:00              │  │
│  │ 이수정 · 친구          │  │
│  │ "한 달 만이에요"        │  │
│  │ [✓ 완료] [⏰ 연기]     │  │
│  └───────────────────────┘  │
│                             │
│  내일 · 5월 5일 (화)         │
│  ┌───────────────────────┐  │
│  │ 🎂 박영호 생일         │  │
│  │ 매년 5/5               │  │  ← Repeat indicator
│  │ [✓ 챙겼어요]           │  │
│  └───────────────────────┘  │
│                             │
│  ─────────────              │
│                             │
│                       [+]   │  ← FAB (리마인더 추가)
├─────────────────────────────┤
│ [홈][검색][+ ][리마인더*][설정]│
└─────────────────────────────┘
```

이번주 / 다가올(2주~1달) / 완료(최근 30일) 탭은 동일 카드 구조.

## 5.7 인터랙션 / 액션

| 요소 | 동작 |
|------|------|
| 탭 변경 | 같은 라우트, 쿼리 파라미터 `?filter=today` |
| [✓ 완료] | 인라인 ContactLog 미니 모달 → 저장 시 status=done + 다음 리마인더 제안 |
| [⏰ 연기] | 5.5 Sheet → next_at += 선택 기간 + status=snoozed |
| [🗑] | AlertDialog 확인 → status=dismissed |
| [편집] | 5.3 Sheet (기존 값 prefill) |
| 항목 탭 | `/p/{person_id}` 인물 상세로 이동 |
| FAB(+) | 5.3 Sheet (인물 미선택 상태) |
| 인물 검색 (생성) | 디바운스 200ms, 최근 사용 5건 우선 |
| 빠른 선택 칩 | 날짜 자동 채움 |
| 반복 (yearly) | 생일/기념일 종류 선택 시 자동 ON |
| 저장 | Reminder 생성 + Web Push 등록 + 토스트 "{날짜}에 알려드릴게요" |
| Push 권한 미허용 | 채널 [Web Push] 옆 노란 점 + "권한 허용 필요" + [허용하기] |

## 5.8 사용 shadcn 컴포넌트

| 영역 | 컴포넌트 |
|------|----------|
| 탭 | `Tabs`, `TabsList`, `TabsTrigger` |
| 카드 | 커스텀 ReminderItem (component-spec.md 참조) |
| 생성 모달 | `Sheet` side="bottom" + `Form` |
| 인물 검색 | 커스텀 Combobox (Command 패턴, 추가 설치) |
| 날짜 선택 | `Calendar`(추가 설치) + `Popover` |
| 시간 선택 | 커스텀 (Input type=time) |
| 토글 칩 | `Badge` 또는 `ToggleGroup` |
| 라디오 | `RadioGroup` |
| 체크박스 | `Checkbox` |
| 연기 Sheet | `Sheet` |
| 삭제 확인 | `AlertDialog` |

## 5.9 데이터 의존성

| 영역 | 쿼리 |
|------|------|
| 오늘 탭 | `Reminder where scheduled_at::date = today and status in (active, snoozed) order by scheduled_at` |
| 이번주 | `where scheduled_at between today and (today + 7d)` |
| 다가올 | `where scheduled_at between (today + 7d) and (today + 30d)` |
| 완료 | `where status='done' and updated_at > (now - 30d) order by updated_at desc` |
| 인물 정보 조인 | `Person where id = Reminder.person_id` |
| 그룹 카운트 | 각 탭 헤더에 표시 (캐싱) |

## 5.10 상태별 처리

| 상태 | 화면 |
|------|------|
| 빈 상태 (각 탭) | 5.2 형식 + 적절한 카피 |
| Web Push 권한 거부 | 상단 배너 "푸시 알림이 꺼져있어요. [설정] " |
| 권한 허용 후 토큰 발급 실패 | 토스트 "알림 등록에 실패했어요. 잠시 후 다시 시도해주세요" |
| 같은 인물 활성 followup 중복 | 생성 시 AlertDialog "기존 리마인더({날짜})를 갱신할까요?" |
| 미래 시점 검증 | 과거 일시 선택 시 인라인 에러 + 저장 비활성 |
| 발송 실패 | 서버에서 재시도 큐 (사용자에게는 노출 안 함) |
| iOS PWA 미설치 | "iOS는 홈 화면에 추가 후 알림이 가능해요" 안내 |

## 5.11 한국어 카피

| 위치 | 카피 |
|------|------|
| 탭 라벨 | "오늘" / "이번주" / "다가올" / "완료" |
| 빈 상태 (오늘) | "오늘은 알림이 없어요" |
