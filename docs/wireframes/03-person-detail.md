## 3.5 탭 구성 (인물 상세 본문)

| 탭 | 표시 | 마일스톤 |
|----|------|----------|
| 타임라인 | 연락로그/노트/이벤트/선물 통합 | M1(연락로그·노트), M2(이벤트·선물·메시지) |
| 메모 | Note 리스트, pinned 우선 | M1 |
| 경조사·선물 | Event + Gift, 합계 표시 | M2 |
| 메시지 | Message 리스트 | M2 |

M1에서는 탭 없이 단일 타임라인 + 메모 섹션. M2에서 `Tabs` 도입.

## 3.6 인터랙션 / 액션

| 요소 | 동작 |
|------|------|
| ← 뒤로 | 이전 화면 (홈/검색/AI 추천) |
| ⋮ 메뉴 | 3.3 Sheet 열기 |
| 프로필 사진 탭 | 풀스크린 이미지 뷰어 |
| 태그 칩 탭 | 검색 결과로 이동 (해당 태그 필터) |
| [💬 연락 기록] | 3.2 Sheet 열기 |
| [🔔 리마인더] | 리마인더 생성 모달 (05-reminder.md 참조) |
| [📝 메모 추가] | 인라인 Textarea 또는 풀스크린 Sheet |
| AI 점수 게이지 탭 | 점수 산출 근거 Dialog |
| AI 사유 우상단 ✕ | "이 추천 별로에요" → AIFeedback 저장 + 카드 미니멀화 |
| 타임라인 항목 탭 | 항목 편집 Sheet |
| 타임라인 항목 스와이프 좌 | 삭제 (확인 후) |
| 타임라인 [+] | 항목 추가 메뉴 (연락/노트/경조사(M2)) |
| 타임라인 필터 ▾ | 종류별 필터 (전체/연락/노트/경조사) |
| 더 보기 | 페이지 30건씩 추가 로드 |
| Deep link `?action=log` | 페이지 진입 0.5초 후 3.2 Sheet 자동 오픈 |

## 3.7 사용 shadcn 컴포넌트

| 영역 | 컴포넌트 |
|------|----------|
| 프로필 사진 | `Avatar`, `AvatarImage`, `AvatarFallback` |
| 태그 / 칩 | `Badge` |
| 액션 칩 | `Button` variant=outline size=sm |
| AI 점수 게이지 | 커스텀 RelationshipScoreGauge (SVG) |
| 연락 기록 모달 | `Sheet` side="bottom" + `Form` + `RadioGroup`(추가 설치) |
| 메뉴 Sheet | `Sheet` side="bottom" |
| 타임라인 항목 | 커스텀 (component-spec.md ContactLogItem) |
| 탭 (M2) | `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent` |
| 삭제 확인 | `AlertDialog`(추가 설치) |
| 점수 근거 | `Dialog` |

## 3.8 데이터 의존성

| 영역 | 쿼리 |
|------|------|
| 프로필 | `Person where id={id} and user_id=auth.uid()` |
| 태그 | `Tag join PersonTag` |
| 점수 | `RelationshipScore where person_id={id}` |
| 타임라인 | `union(ContactLog, Note, Event(M2), Gift(M2)) where person_id={id} order by occurred_at desc` |
| 다음 리마인더 | `Reminder where person_id={id} and status='active' order by scheduled_at limit 1` |

## 3.9 상태별 처리

| 상태 | 화면 |
|------|------|
| 로딩 | 스켈레톤 (헤더 + 게이지 + 타임라인 5개) |
| 타임라인 빈 상태 | "아직 연락 기록이 없어요. 첫 기록을 남겨볼까요?" + [💬 연락 기록] |
| AI 점수 미생성 (5명 미만 등) | 게이지 자리에 "아직 분석할 데이터가 부족해요" |
| 인물 미존재 / 권한 없음 | 404 페이지 → 홈 복귀 안내 |
| 삭제 진행 중 | "삭제 중..." 토스트 + Optimistic 화면 사라짐 |
| 사진 로드 실패 | `AvatarFallback` (이름 첫글자) |
| 네트워크 에러 | 에러 카드 + [다시 시도] |

## 3.10 한국어 카피

| 위치 | 카피 |
|------|------|
| 액션 칩 | "💬 연락 기록" / "🔔 리마인더" / "📝 메모 추가" |
| 빠른 액션 | "방금 연락함" |
| AI 사유 헤더 | "AI 관계 건강도" |
| 점수 라벨 | 0~39 "주의" / 40~69 "보통" / 70~89 "양호" / 90+ "매우 좋음" |
| 빈 타임라인 | "아직 기록이 없어요. 첫 연락을 기록해볼까요?" |
| 삭제 확인 | "이 인물을 삭제하면 관련 연락 기록·리마인더도 함께 삭제됩니다." |
| 미래 일시 | "미래 시점이에요. 그래도 기록할까요?" |
| 5분 내 중복 | "방금 기록한 내용과 합칠까요?" |
| 다음 리마인더 토스트 | "기록되었어요. 다음은 {날짜}에 알려드릴게요." |
