## 9.9 인터랙션 / 액션

| 요소 | 동작 |
|------|------|
| 다크 모드 토글 | 즉시 적용 + localStorage + `<html class="dark">` |
| Web Push 토글 | 권한 요청 / 해제 (브라우저 API) |
| AI 프로바이더 변경 | 즉시 저장, 다음 분석부터 적용 |
| Export 버튼 | 1만 행 이내: ZIP 즉시 다운로드, 초과: 이메일 발송 + 토스트 |
| Import (P1) | 파일 선택 → 매칭/병합 UI |
| 캐시 비우기 | IndexedDB + Service Worker 캐시 삭제 + 재로드 |
| 휴지통 [복원] | deleted_at = null + 토스트 |
| 휴지통 [영구 삭제] | AlertDialog 이중 확인 → hard-delete |
| 표시 이름 변경 | 즉시 저장 + 토스트 |
| 이메일 변경 | 비밀번호 재입력 → 새 이메일 인증 메일 발송 |
| 비밀번호 변경 | 현재 비밀번호 재입력 → 새 비밀번호 + 강도 체크 |
| 로그아웃 | AlertDialog "정말 로그아웃하시겠어요?" + [로컬 캐시 함께 삭제] 옵션 |
| 회원 탈퇴 | 9.7 화면 → 30일 유예 후 hard-delete |
| 테스트 알림 | 즉시 푸시 발송 (디버깅용) |

| 영역 | 컴포넌트 |
|------|----------|

## 9.11 데이터 의존성

## 9.12 상태별 처리

| 상태 | 화면 |
|------|------|
| Web Push 거부 (브라우저) | 토글 OFF + 안내 "브라우저 설정에서 허용해주세요" + 가이드 링크 |
| iOS PWA 미설치 | 알림 섹션 상단 배너 "iOS는 홈 화면에 추가 후 알림이 가능해요" |
| 사용량 80%+ | 진행 바 노란색 + 경고 |
| 사용량 100% | 진행 바 빨간색 + Quota 도달 안내 |
| Export 진행 중 | "내보내기 준비 중... 곧 다운로드됩니다" 토스트 |
| Export 실패 | 에러 토스트 + 재시도 |
| 휴지통 비어있음 | "휴지통이 비어있어요" 빈 상태 |
| 캐시 비우는 중 | 로딩 + "캐시를 비우는 중이에요..." |
